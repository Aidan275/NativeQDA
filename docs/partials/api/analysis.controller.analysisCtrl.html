analysisCtrl
controller in module analysis

Description

Controller for the analysis page.

Methods

  • activate()

    Runs when the page first loads and starts the loading overlay for the analysis table.

  • confirmDelete(name, id)

    Displays a confirmation message to confirm deletion of the passed analysis object

    Parameters
    ParamTypeDetails
    namestring

    Name of the analysis object

    idstring

    ObjectId of the analysis object

  • deleteAnalysis(name, id)

    Uses the deleteWatsonAnalysis function from analysisService to delete the analysis object from the database.

    Parameters
    ParamTypeDetails
    namestring

    Name of the analysis object

    idstring

    ObjectId of the analysis object

  • getAnalysesList()

    Uses the listWatsonAnalysis function from analysisService to load the analysis objects from the database.

  • listAnalyses()

    Lists the analysis objects in a table using the NgTable directive.

  • popupNewAnalysis()

    Displays the newAnalysisCtrl modal.

  • removeFromList(id)

    Removes the deleted analysis object from the local analysis array and updates the table .

    Parameters
    ParamTypeDetails
    idstring

    ObjectId of the analysis object
