radarChartCtrl
controller in module visualisations

Description

Controller for the radar chart visual.

Methods

  • activate()

    Runs when the page first loads and uses the readWatsonAnalysis function from analysisService to load the analysis results from the database for the given analysis id. Then structures the data and calls the function to draw the radar chart.

  • toggleOptions()

    Toggles the slideout options side bar

  • toogleAllBoxes()

    Checks/unchecks all the entities/keywords checkboxes by toggling the checked boolean for all the data in the @linkvm.analysisData array then re-draws the radar chart.

  • updateRadarChart()

    Re-draws the radar chart depending on which checkboxes are checked.
