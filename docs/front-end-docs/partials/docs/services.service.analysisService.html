analysisService
service in module services

Description

This is an angular service used for making requests to the server to perform and read analyses.

Methods

  • deleteWatsonAnalysis(analysisId)

    Makes a DELETE request to the back-end to delete the specified Watson analysis object.

    Parameters
    ParamTypeDetails
    analysisIdString

    The ObjectId for the analysis object

  • listWatsonAnalysis()

    Makes a GET request to the back-end to get all the Watson analysis objects.

    Returns
    Array

    An array of the analysis objects

  • readWatsonAnalysis(analysisId)

    Makes a GET request to the back-end to get one specified Watson analysis object.

    Parameters
    ParamTypeDetails
    analysisIdString

    The ObjectId for the analysis object

  • readWatsonCategories(analysisId)

    Makes a GET request to the back-end to get the category results of the specified Watson analysis object.

    Parameters
    ParamTypeDetails
    analysisIdString

    The ObjectId for the analysis object

  • saveWatsonAnalysis()

    DEPRECATED - Analysis is now saved directly to the database after being performed

  • watsonAnalysis(data)

    Makes a POST request to the back-end to perform and save a Watson analysis.

    Parameters
    ParamTypeDetails
    dataObject

    An object consisting of:

    url: url to the file to be analysed (text file on S3)
    name: name of the analysis given by user
    description: description of the analysis given by user
    createdBy: user's name
    sourceDataKeys: array of the S3 keys used

    Returns
    Object

    The analysis object

  • watsonTextAnalysis(data)

    Makes a POST request to the back-end to perform and save a Watson analysis.

    Parameters
    ParamTypeDetails
    dataObject

    An object consisting of:

    text: string to be analysed
    name: name of the analysis given by user
    description: description of the analysis given by user
    createdBy: user's name
    sourceDataKeys: array of the S3 keys used

    Returns
    Object

    The analysis object
