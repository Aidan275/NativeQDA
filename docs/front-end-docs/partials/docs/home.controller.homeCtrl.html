homeCtrl
controller in module home

Description

The main homepage/dashboard of the application. Displays a small map and the recently uploaded files.

Methods

  • activate()

    Runs when the page first loads and starts the loading overlay for the map/file-list and calls the initMap function.

  • addMapMarkers()

    Adds the files to the map, checking each file type and using the appropriate marker icon. Compiles and binds each the popup window and adds relevant event calls for each marker.

  • geoLocateUser()

    Geolocates the user. If locationfound, calls onLocationFound to update the map with the user location. if locationerror, calls onLocationError

  • getFileList()

    Uses the getFileList function from filesService to load the files from the database. On success, calls addMapMarkers to add the files to the map and the recent files table.

  • initMap()

    Initialises the map setting up the initial settings such as default location, zoom level, map tiles position marker, and sidebar. Next the functions to get the users location and the file markers from the database are called.

  • listFiles()

    Lists the 7 most recent files in the Recent Files table

  • onLocationError(error)

    Displays error message to the user

  • onLocationFound(response)

    Adds position marker to the map at the users location, adds a accuracy raduis, and a binds a popup with with the radius distance.
