surveyEnterCodeCtrl
controller in module survey

Description

This controller checks if a survey code is valid before forwarding the user to the complete survey page, where the survey is displayed.

The system checks if the IP address of the user has recently completed the given survey, if so, the IP address is locked out from the given survey for 5 minutes to prevent people from entering false/inaccurate data multiple times.

Dependencies

  • $location
  • surveyService
  • logger
