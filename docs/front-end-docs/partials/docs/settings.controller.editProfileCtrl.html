editProfileCtrl
controller in module settings

Description

This is a popup modal where a user can modify their profile information, including:

  • Uploading an avatar image
  • Editing their name
  • Editing their email
  • Editing their company name
  • Editing their password

Dependencies

  • $scope
  • $uibModalInstance
  • $animate
  • bsLoadingOverlayService
  • Upload
  • usersService
  • authService
  • s3Service
  • logger
