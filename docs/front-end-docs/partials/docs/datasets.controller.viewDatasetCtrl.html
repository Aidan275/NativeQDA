viewDatasetCtrl
controller in module datasets

Deprecated API Was added as we anticipated that analyses would have settings that could be configured per analysis, so instead of needing to select multiple files each time the settings of an analysis were changed, we added the concept of datasets which consisted of a selected number of files. This has not happened yet so datasets only add an unnecessary step to the analysis process.

Description

Opens a popup modal and displays the details of the selected dataset.

Dependencies

  • $uibModalInstance
